## getFactions

Return a paginated list of all the factions in the game.

### Example Usage

```typescript
import { SeTypescriptSDK } from "se-spacetraders-typescript-sdk";

async function run() {
  const sdk = new SeTypescriptSDK();

  const page = 1;
  const limit = 10;
  
  const result = await sdk.factions.getFactions(page, limit);

  // Handle the result
  console.log(result)
}

run();
```

## getFaction

  const factionSymbol = "COSMIC";
